Badly written python script for controlling my pioneer PDF-1009

Code runs on a raspberry pi with an IR-Header

Effectively lets you create a .db file of your CD library and then search it
create .txt print outs, add and remove titles from it, create and playback playlists.
Accomplishes this by converting your selected songs to a series of equivalent IR instructions
which the player then reads.

Also includes a manual remote control for some other HiFi I was using at the time.
Hope it brings you some joy :)
